# Standalone Web Tool Template (最もシンプルな Hello World ツール)

Free Web Tools の新規ツールを開発するための標準テンプレート（Scaffolding）です。

---

## 1. 構成とファイル

* **`index.html`**: ポータルにマウントされた状態で動作することを確認するための親 HTML。
  * 共通 CSS/JS を自動で動的にロードする「環境自動判定型アセットローダー」を搭載しており、ローカル起動（`file:///`）とサーバー起動（`http:` 等）を自動判定して動作します。
  * CORS 制限時でもタイトルが `page_title` に化けないための自動フォールバック機能が組み込まれています。
* **`dev.html`**: 完全に独立した単体開発用の HTML。
  * 外部の共通 CSS/JS やローカルサーバーに一切依存せず、ブラウザでダブルクリックするだけで直ちに動作します。
* **`js/workspace.js`**: ツール本体の Vue コンポーネント `ToolWorkspace` の定義ファイル。
  * ツール独自の UI マークアップ（`template`）と JavaScript ロジック（`setup`）はすべてここに記述します。
* **`i18n.json`**: 英語（`en`）と日本語（`ja`）の多言語翻訳定義ファイル。

---

## 2. 開発手順 (How to Develop)

新しいツールを開発する際は、以下のステップで行います。

### Step 1. コピー＆リネーム
この `tool-template` ディレクトリを丸ごとコピーし、`tools/my-new-tool` などのディレクトリ名で新しいツールを作成します。

### Step 2. `dev.html` を用いたスタンドアロン開発
1. `dev.html` をエディターとブラウザ（直接ファイルを開く）で起動します。
2. `js/workspace.js` を書き換えて、ツールのロジックやUIを実装します。
3. `i18n.json` に必要な翻訳テキストを追加します。
4. `dev.html` をブラウザでリロードして動作を確認します。

### Step 3. `index.html` でのポータル連携確認
1. 実装が完了したら、`index.html` を開いて動作を確認します。
2. 共通アセットが正しく動的にロードされ、ヘッダー・フッター・右下コントロール、設定モーダルおよび背景画像変更がシームレスに機能することを確認します。

---

## 3. テンプレート（Hello World）の仕様
本テンプレートは初期状態で **「最もシンプルな Hello World & クリックカウンター」** が実装されています。
* 言語設定（日・英）の切り替えに応じて、メッセージがリアクティブに変更されます。
* Vue のもっとも基本的な状態（`ref`）とアクション（`@click`）がバインドされ、カウンター機能の動きをひと目で確認できます。
